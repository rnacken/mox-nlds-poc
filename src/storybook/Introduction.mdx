    # Introductie

    Het Mox design system is een implementatie van het NL DesignSystem (NLDS).
    We onderscheiden ons van de huidige implementaties (zoals Rijkshuisstijl Community) door:
    - Een [**atomic system**](/docs/atomic-system--docs) (**Tokens < CSS Atoms < React Atoms < React Molecules < React Organisms**) voor hierarchie tussen componenten
    - Consistentere APIs (mogelijke properties) voor React **Atoms** (en typesafe!)
    - Een kleinere set `tokens`, enkel voor de CSS **Atoms**
    - Een [**scaling system**](/docs/scaling-system--docs) voor zowel **typografie** als **spacing** (bv `font-size`, `gap`, `padding`, etc), waarbij de sizes (bv `sm`, `md`, `4xl`, etc) met `clamp` relatief meegroeien met je schermgrootte
    - Een set property-classNames, waarvan sommige responsive zijn (bv `inlineSize-md@mobileMin` om de `inlineSize` op `md` zetten bij mobile-screens en groter)
